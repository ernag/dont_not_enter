This folder contains the Keil uVision 4 multi-project workspace file which can be used to batch-build the MQX kernel and all other MQX software stacks and libraries.

Open the build_libs.uvmpw file in the uVision 4 environment and go to the Project / Batch Build menu. You will be able to mass-build all or selected library projects in both Debug and/or Release configuration.

Remember to rebuild all MQX libraries after any change made to the MQX configuration in the user_config.h file. Read more information about the MQX in the Release Notes and Getting Started documents.
